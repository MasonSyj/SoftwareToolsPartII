缅怀老校长

大概去年这个时候，老校长逝世，十分惋惜。还记得刚进学校的时候，无意中听说了“老校长”。我第一次听到的时候心里有些纳闷，学校那么多年的历史，应该也有很多校长了，怎么偏偏还有一位校长会被如此称呼。

时不时在课堂上也会听到老师说老校长，讲到那个时代，老校长的治学理念，而那个时代的学校氛围和风气，就感觉挺向往的，如今在大学里种种桎梏限制着学生的自由。但也是过了很长一段时间，在图书馆借了本老校长的生本回忆录。才相对完整的了解了老校长的一生，他的求学之路、在动荡时代的漂泊、治校经历。同时最重要的，老校长一生躬于学问。

不同于读其他人的回忆录，读老校长的就觉得十分亲切和朴素，比如在高中因为言论被退学；比如在武汉的军营因为说了不该说的话被关监禁；比如叹息黄鹤楼长江大桥的建造而迁址。最令我印象深刻而有趣的事，是在当时，老校长想要把文学院楼里的毛泽东像搬走，便利用周五师生都在露天电影场看电影的时候，不仅免收了5分的电影票钱，而且加播了一场电影。

虽然没有与老校长接触的机会，但是感觉到学校曾经有这么一位优秀杰出的校长，感觉十分敬仰。也感觉到了很多地方值得去自己学习，自己还年轻。也借此次机会，缅怀老校长。
